キャンペーンモジュールでは、Brightcoveプレーヤーの内部でビデオが再生されるときにビューア情報をキャプチャできるカスタムリードフォームを作成できます。リードデータは Salesforce に送信され、Salesforce によって処理されます。

カスタムリードフォームを作成するには、次の手順を行います。

  1. Salesforce でフォームを作成する
  2. フォームをコピーして編集する HTML
  3. キャンペーンモジュールにカスタムリードフォームを作成する
  4. リードフォームをプレーヤーに割り当てる
  5. Salesforce でフォームデータを検証する
注:視聴者の情報を取得し、データを Video Cloud に保存する標準のリード フォームについては、次を参照してください。キャンペーン リード フォームの作成.

Salesforce でのフォームの作成

Salesforce の Webto-Lead 機能を使用すると、Web サイトの訪問者からリード情報を生成できます。これらのフォームは、Brightcoveプレーヤーの内部に表示することもできます。以下の手順は、名、姓、メールアドレス、会社、都市、州/省を取得するための簡単なフォームを作成します。Web-to-リード機能の詳細については、セールスフォースのドキュメント.

  1. Salesforceアカウントにログインします。
  2. から設定、 入力Web-to-リード[クイック検索] ボックスで、 Web-to-リード.
  3. クリックWeb-to-リードフォームの作成.
  4. Web-to-Lead フォームに含めるフィールドを選択します。[追加] 矢印と [削除] 矢印を使用して、フィールドを利用可能なフィールドリストと選択したフィールドリストを表示し、上矢印と下矢印を使用してフォームのフィールドの順序を変更します。
  5. 入力しますリターン URL .この例では、返品 URL は http://www.brightcove.com に設定されます。キャンペーンモジュール内でフォームをカスタムリードフォームとして使用する場合、この値は無視されます。
  6. クリック生成.フォーム HTML が表示されます。

フォームのコピーと編集 HTML

キャンペーンモジュール内のフォームを使用するには、フォームHTMLをコピーする必要があります。また、フォームに基本的なスタイリングを行うこともできます。以下の手順では、HTMLエディタにフォームHTMLをコピーし、フォームにいくつかのスタイル変更を行います。

デフォルトでは、フォームは　Brightcove Player内部で表示されると、プレーヤー上の透明な背景に表示されます。背景色を設定するには、CSS を使用します。

  1. HTML をクリップボードにコピーします。
  2. クリック終了した.
  3. フォームHTMLを、お気に入りのHTMLエディターに貼り付けます。エディターのコード整形機能を使用して、読みやすいようにコードを整形したい場合があります。
  4. HTMLエディタを使用して、次のCSSを追加して背景色をグレーに変更します。
    <風格>
    ボディ {
    背景色:rgba (204,204,204,0.9);
    マージン:0;
    パディング:20px;
    高さ:100%;
    }
    </ style>
  5. (オプション) Campaign によって生成されたリードを識別するために、次のコードを<フォーム>鬼ごっこ：
    <inputtype="hidden"name="lead_source"value="ビデオクラウドキャンペーン">
  6. HTML をクリップボードにコピーします。

キャンペーンモジュールでのカスタムリードフォームの作成

SalesforceでHTMLフォームが生成されたら、キャンペーンモジュールにカスタムリードフォームを作成し、そのリードフォームをキャンペーン対応プレーヤーに関連付けます。

  1. を開きます運動モジュール。
  2. クリックリードフォーム左のナビゲーションにあります。リードフォームが作成されている場合、作成されたリードフォームが表示されます。
  3. クリックリードフォームを作成.
  4. リードフォームに説明を付ける名前.
  5. を選択フォーム言語.
  6. を選択フォームのタイミング.フォームのタイミングで、リードフォームがいつ表示されるかが決まります。
  7. フォームの種類をカスタムリードフォーム.
  8. に貼り付けますフォーム HTMLセールスフォースから。
  9. (オプション)に貼り付けます確認 HTML .確認ビューが5秒間表示され、その後、動画の再生が再開されます。確認HTMLが含まれない場合は、フォームの送信後に動画の再生が再開されます。
  10. (オプション)クリックプレビューフォームフォームのプレビューを開きます。
  11. クリック保存リードフォームを保存します。

リードフォームをプレーヤーに割り当てる

作成したリードフォームは、プレーヤーに割り当てる必要があります。リードフォームをプレーヤーに割り当てるには、次の手順を行います。

  1. クリックキャンペーンプレイヤー左のナビゲーションにあります。
  2. プレーヤーを見つけて使用しますリードフォームプルダウン リストからリードフォームを選択します。

これでセットアップは完了しました。リード情報を取得する準備ができました。キャンペーン対応プレーヤーを使用してビデオを公開すると、フォームのタイミング設定に従ってリードフォームが表示されます。

Salesforce でのフォームデータの検証

カスタムリードフォームを使用してキャプチャされたリードは、Salesforce アカウントのリードとして追加されます。
