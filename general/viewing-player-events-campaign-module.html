キャンペーンモジュールは、ビューイベントを追跡する機能を提供します。ビューイベントがキャンペーンモジュールに送信される頻度は、ビデオの長さに基づいて動的に計算されます。キャンペーンは、4分の1回につき1回、最低4つのイベントで送信されます。

注:ビュー イベント データは、データのエクスポートオプションまたはオーディエンス API .

ビュー イベントを表示するには、キャンペーン モジュールを開き、最近のすべてのアクティビティ左のナビゲーションにあります。このページには、過去 24 時間の表示イベントアクティビティが表示されます。

のインタラクション イベントを表示リンクは次の場合に有効になりますカスタム イベントまたは指標ビュー イベントに添付されます。

このページには、次の情報が表示されます。

  • タイムスタンプ- ビデオ視聴の日時
  • 動画名とID - ビデオ名と Video Cloud ビデオ ID
  • 視聴率- 視聴者が動画の視聴を停止したポイントを動画の長さで割った値。たとえば、視聴者が 2 分間の動画の最初の 5 秒を視聴した後、55 秒ポイントまでスキップし、さらに 5 秒視聴してビデオを停止（またはページを出た）場合、視聴率は 50% として記録されます。
    注:為にライブ動画、キャンペーンで視聴された割合を追跡することはできません。Campaignモジュールは、それがどのくらい続くかを知ることができないため、監視された合計時間だけを追跡します。
  • 視聴時間- 総視聴時間
  • 外部 ID - ビューアを識別するために使用される値。たとえば、Eloqua GUID、Marketo GUID、HubSpot GUID、SSO GUID、またはビューアを識別するために使用された別の一意の値。
  • 状態- イベントのステータス
    • 未処理- イベントがマーケティング オートメーション プラットフォームに同期されていない
    • 同期された- イベントはマーケティング オートメーション プラットフォームに正常に同期されました
    • sync_disabled - キャンペーン モジュールが同期を無効にして構成されていたため、イベントがスキップされました
    • 失敗した- エラー同期;ツール ヒントには、発生したエラーに関する詳細が表示されます
    • 不明な連絡先- 不明なユーザーのため、イベントは同期されませんでした
    • なし- 適用できない;カスタム、キャンペーン トラッキング、または Marketo Munchkin 統合を使用している場合に表示されます

ページの上部にあるドロップダウンリストを使用して、イベントデータをステータス別にフィルタリングすることもできます。

インタラクションイベントの表示

カスタム メトリックまたはイベントがビュー イベントに関連付けられている場合、インタラクション イベントを表示リンクが有効になります。たとえば、キャンペーン対応プレーヤーでギャラリー ページはめ込みエクスペリエンスを使用していて、そのエクスペリエンスが次のように構成されているとします。リンク、カード、その他のインタラクティブな要素の場合、エクスペリエンスはインプレッション イベントとコンバージョン イベントを Campaign モジュールに送信します。これらのイベントは、Campaignカスタムフィールドを使用してCampaignに渡されます。イベント データは、キャンペーン モジュールの一部として表示されます。最近のすべてのアクティビティ.

注:使用時に相互作用イベントも表示される場合がありますキャンペーンのカスタム指標とイベント.

次のデータが渡されることがあります。

  • イベント1 - 視聴者が操作する機会があった CTA 名のカンマ区切りリスト (インプレッション)
  • イベント 2 - 視聴者が使用する CTA 名のカンマ区切りリストやりましたインタラクション (変換)
  • メトリック 1 - CTA インプレッションの総数
  • メトリック 2 - CTA コンバージョンの合計数 (リンクのクリック、フォームの送信など)

イベントを表示するには、インタラクション イベントを表示ヘッダーにリンクします。

その他のデータ表示オプション

キャンペーンデータは、次の方法で表示/エクスポートできます。

  • ビデオ活動- ビデオごとに視聴者のアクティビティを表示するために使用されます。詳細については、を参照してください。ビデオ アクティビティの表示.
  • 視聴者プロフィール- 視聴者によるビデオ アクティビティの表示に使用されます。詳細については、を参照してください。視聴者のプロフィール情報を表示する.
  • データのエクスポート- ビデオ ビューとリード データのエクスポートに使用されます。詳細については、を参照してください。キャンペーン モジュールからのキャンペーン データのエクスポート.
  • オーディエンス API - Audience API を使用して、ビデオ ビューとリード データをプログラムで取得できます。詳細については、を参照してください。概要：オーディエンス API .

よく寄せられる質問

ViewイベントはサードパーティのMAP接続とどのように同期されますか。
同じビデオ再生セッションに複数のレコードがある場合があります。セッションIDが同じであっても、15分後にアクティビティが受信されない場合、キャンペーンは新しい'Viewイベント'を考慮します。また、MAPデータは1時間おきに同期されます。
