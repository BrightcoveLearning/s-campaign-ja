キャンペーンの理由 & 仕組み

マーケティングオートメーションソリューションで動画を追跡する必要があるのはなぜですか？
マーケティング自動化ソリューションは、今日の動画を追跡しません。強力なコンテンツとして動画に投資する場合は、その効果を追跡し、コンテンツを閲覧したリード/ユーザーとのコミュニケーションを自動化する必要があります。
BrightcoveとCampaignモジュールはクッキーを利用していますか?
キャンペーンモジュールは、統合が行われているときにサードパーティのcookieを作成し、使用してユーザを識別します。この方法では、視聴者は 2 回目にリードフォームへの記入を求められることはありません。このモジュールでは、Marketo、HubSpot、Eloqua API も利用し、クッキーを使用してビューアを追跡します。キャンペーンプロファイルを使用する場合は、Brightcoveクッキーが使用されます。
Brightcoveキャンペーンはすべてのお客様に対して有効になっていますか?
いいえ。VideoMarketingSuite(VMS)の購入可能なアドオンとしてキャンペーンをご利用いただけます。ただし、キャンペーンプロファイルはBrightcoveのすべてのお客様が無料で利用できます。キャンペーンライトだと思ってください。
キャンペーンプロファイルの対象者
2017年第4四半期から、すべてのVMSのお客様が無料でキャンペーンプロファイルにアクセスできるようになります。 データの追跡を開始するには、顧客がそれを有効にする必要があります.EnterpriseVideoSuite（EVS）のお客様は、キャンペーンSSOトラッキングも利用できます。

視聴者の追跡と制限

キャンペーンプロファイルには、フル機能のキャンペーンモジュールと比べてどのような制限がありますか。
  • サードパーティ製システムとの統合は不要
  • APIアクセスなし
  • カスタムリードフォームなし（キャンペーンリードフォームのみ）
  • 1000 ビュー/最大 24 時間キャプチャ時間
キャンペーンデータをCSVまたはXLSファイルに手動でエクスポートすることもできます。
マーケティング自動化プラットフォームに接続する場合、キャンペーンにはどのような制限/要件がありますか?
制限と要件については、各プラットフォームのドキュメントに記載されています。
  • オラクル・エロクア
  • ハブスポット (クライアント側 APIまたREST API )
  • マルケト (マンチカンまたREST API )
  • セールスフォース
オーディエンス API をポーリングできる頻度に制限はありますか？動画検索結果に1000のキャップがあることは知っていますが、そのAPIエンドポイントには他の周波数関連のキャップがありますか？
API 呼び出しあたりのビューイベントの最大数は 100 です。レート制限はありませんが、あまりにも多くを呼び出そうとすると、おそらく少し遅くなります。
キャンペーンデータを現在の3時間遅延より早く取得する方法はありますか?私は'リアルタイムに近い'指標を取得したいと思います。
なしリアルタイムメトリクスが必要な場合は、独自のプラグインを作成し、データを自分のシステムに送信する必要があります。
イベントビューがキャンペーンに送信されるトリガーは何ですか。ビデオが完了したとき、プレーヤーウィンドウが閉じられたとき、または何か他のものですか？ユーザーがビデオを完了せず、プレーヤーウィンドウを閉じない場合はどうなりますか？
さまざまですが、基本的にキャンペーンプラグインは10秒ごとにハートビートをサーバに送信します。
キャンペーンでは、完了するビューを追跡することを検討しているのはいつですか?プレーヤーウィンドウが閉じているときですか、ビデオが完成したときですか？彼らが間近で見て、ビデオを一時停止し、プレーヤーウィンドウを閉じない場合はどうなりますか？
私たちが最後にハートビートを見たから15分が経過した時です。基本的に活動が不足しています。
Campaignビューイングデータで視聴者データが入手できたら変更することは可能ですか?そのイベントデータは、将来何らかのログ統合によって更新されますか？
UIまたはAPIを介してキャンペーンでデータが利用可能になると、データは'決済済み'と見なされ、変更されることはありません。

データ保護と記録保持

キャンペーンGDPRに準拠していますか?

はい、キャンペーンでは、キャンペーンを通じて収集したすべてのデータを検索および消去し、ユーザーの代理として行動することができます。

https://studio.support.brightcove.com/campaign/admin/managing-viewer-data.html

キャンペーンでデータを保持できる期間はどれくらいですか?
MAP 統合は最大 6 か月に設定でき、非 Map 統合は 3 年に設定できます。
GUID ルックアップがキャッシュされる期間はどれくらいですか
既知の場合は 30 日、不明の場合は 7 日間 (MAP 統合にのみ適用され、非 Map 統合にはルックアップは必要ありません)

マーケティング自動化とビューア分析

マーケティングオートメーションプラットフォームの他のメディアと動画のトラッキングについて何が違うのですか？
ウェブページやダウンロード可能なホワイトペーパーとは異なり、ビデオを使用すると、視聴者が視聴したビデオの量、つまりどのコンテンツが公開されているかを正確に判断できます。
Eloqua/marketo/HubSpotアカウントとどのように接続しますか？
Eloqua/Marketo/HubSpotアカウントには、キャンペーンモジュールのVideoMarketingSuiteから直接ログインできます。そこから、MAPトラッキングを内蔵したプレイヤーとリードフォームを作成できるため、動画分析のトラッキングをすぐに開始できます。
Eloqua/marketo/HubSpotページだけでなく、非Eloqua/marketo/HubSpotのランディングページ（サイトコア/ワードプレス/ギャラリーなど）のビューを追跡できますか？
はい。使用中のプレイヤーがキャンペーン有効（キャンペーンプラグインがインストールされている）である限り、プレイヤーがMAPランディングページにあるか、他のランディングページにあるかにかかわらず、すべてのビューが追跡されます。
この統合で追跡できるものは何ですか？
当社の統合により、データベースにすでに存在する既知の連絡先の動画視聴分析（動画、動画 ID、閲覧率、視聴時間）を追跡できます。
不明な連絡先はどうですか？彼らも追跡できますか？
BrightcoveCampaignを使用すると、不明な連絡先を追跡し、Eloqua/HubSpotフォームを送信すると（知られるようになる）、キャンペーンは視聴者情報をその新しい連絡先に遡って関連付けることができます。キャンペーンは、不明な連絡先のビューデータをOracleEloqua/HubSpotインスタンスに照合するために、視聴情報を毎日再送信します。
私のビデオビューのデータは、マップ内のどこに表示されますか？
これは、使用している MAP によって異なります。については、次のドキュメントを参照してください。オラクルエロクア、マルケトとハブスポット.
Salesforce の動画視聴データはどこに表示されますか？
ビデオビューは、[アクティビティ] セクションのビューアの連絡先の下にタスクとして Salesforce に配信されます。見るSalesforce での動画データの表示これの例として。
連絡先の 1 つが動画を視聴したときに Salesforce に通知を受けることはできますか？
直接じゃないただし、連絡先の活動を通知するレポートを実行することはできます。
SalesforceとOracle Eloqua/Marketoとの追跡に違いはありますか？
最大の違いは、Salesforce がユーザを追跡するために Cookie を使用しないことです。したがって、ビューアを識別するために、一意の識別子を渡す必要があります。見るこのドキュメント詳細については。さらに、CRM である Salesforce は、自動化されたフォローアップやリードスコアリングを提供していません。ただし、Salesforce で ToDo を作成し、実際のフォローアップのために営業チームに割り当てることができます。
キャンペーンにはカスタム統合オプションがあります。キャンペーンはこれを使用して他のどのプラットフォームと統合できますか?

キャンペーンは、次のようなデータプラットフォームと統合できます。

  1. 各連絡先レコードに一意の識別子を使用します。
  2. サードパーティのプラットフォームから追加データを受信できるAPIを持っている(キャンペーンなど)
AnalyticsモジュールのビデオビューがCampaignモジュールのビュー数より多いのはなぜですか?

分析 & キャンペーンモジュールは2つの個別のシステムであり、それぞれが個別のユースケースをキャプチャするため、ビューイベントが1対1であることは想定できません。

キャンペーンのリードフォームにはvideo_viewリードフォームを表示する前にトリガーされたイベント。これらはvideo_viewイベントは、ユーザーが手動で [再生] ボタンをクリックして再生を開始したときにトリガーされます。その結果、コンテンツが再生されたように見えなくても、 video_viewが登録されており、Analytics モジュールの動画に表示されます。

Analyticsモジュールのビデオビューの数がCampaignモジュールのイベントのビューよりも多い場合、最も可能性が高いのは、ユーザがリードフォームを提示されたが、再生を続行するためにそれに記入しなかったためです。

プレイヤーリードフォーム

キャンペーンにリードフォームが表示されるのはいつですか。
デフォルト設定では、MAP データベース内の不明なユーザー (既知のユーザーに変換するユーザー) へのリードフォームのみが表示されます。リードフォームを常に表示するように選択した場合は、設定でこれを上書きできます。これはテストに最も役立ちます。既知のユーザーは自動的に追跡され、そのデータは MAP 内の連絡先に関連付けられ、フォームを表示する必要はありません。
リードフォームをカスタマイズできますか？
UI を使用して、表示されるテキストとフィールドを調整できます。UI で利用可能なものを超えて変更することはできません (ギャラリーでは、フォームがサイトと一致するようにスタイルを更新できます)

一般的なトピック

どの埋め込みコードを使うべきですか？iframeかページ内？
キャンペーンが機能するには、Advancedembedcodeを使用する必要があります。標準（iframe）埋め込みコードが使用されている場合、プレイヤーはページから分離されます。つまり、キャンペーンプラグインは一意の識別子を参照できないため、すべてのユーザは不明と見なされます。
マーケティング自動化とは何ですか？
マーケティング自動化は、さまざまなオンラインマーケティングチャネル（電子メール、ウェブ、フォーム、コンテンツ、ビデオ、ソーシャルなど）とのリード/ユーザーのインタラクションを追跡する技術です。マーケティング自動化により、マーケティング担当者はこれらのチャネル全体でより効果的にマーケティングを行い、反復的なマーケティングタスク（電子メール通信など）を自動化できます。マーケティング自動化により、マーケティング担当者は事前定義された変数に基づいて、自動化されたパーソナライズされたタスクを設定できます。たとえば、ユーザーが特定の動画の 75% を視聴した場合、その特定の件名に関するメールを送信しますが、12% しか視聴していない場合は別のものを送信します。
Brightcoveはマーケティングオートメーション企業ですか？
いいえ、Brightcoveは動画マーケティングのための完全なソリューションを提供しています。当社のソリューションにより、動画をオンラインで入手し、コンテンツを整理、プレーヤーの作成、ビデオポータルの作成、分析の追跡、既存のマーケティング自動化プラットフォームとの統合を行うことができます。Brightcoveは、動画分析をマーケティングオートメーションプラットフォームに統合します。
MarketOカスタムアクティビティが正しく設定されているかどうかを確認するにはどうすればよいですか？
Campaign には、特定の種類の統合用のデバッグ ツールを提供する隠し URL があり、Marketo REST はその 1 つです。 https://studio.brightcove.com/products/videocloud/campaign/debug
カスタムインテグレーションを構成するには何が必要ですか？
3つのことを行う必要があります：
  1. キャンペーンがキャプチャできるページのURLまたはコードに一意の識別子を表示する必要があります。
  2. Audience API を使用してデータを JSON ファイルとして取得するスクリプトを作成します（Brightcove グローバルサービスがこれを支援します）。
  3. データを宛先データベースのフィールドにマップします (スプレッドシートの結合を考えて)。
次のサポートドキュメントでは、これらのタスクについて説明します。
  • Campaign モジュールとのカスタム統合の作成- Campaign が取得できる固有の ID を提供する方法について説明します
  • オーディエンス API - プログラムですべてのデータを JSON ファイルとして取得し、選択したデータ処理システムにインポートする方法について説明します
キャンペーンはどのようにCDOデータをEloquaにプッシュするのですか?
カスタム オブジェクト バルク API を使用します。 https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/api-bulk-2.0-customobjects.html
キャンペーンは外部資産活動をどのように推進しますか？キャンペーン、外部資産および活動タイプを使用する方法を制御できますか。
外部アクティビティ API ( https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/op-api-rest-2.0-data-activity-post.html ）。キャンペーンの名前、アセット名、タイプを制御することはできません。私たちは、独自のキャンペーン（BrightCoveVideoPlay という名前の）を作成し、アクティビティの作成に進みます。BrightCoveVideoPlay という名前もあります。
キャンペーンがサービスXと通信できるようにホワイトリストに登録できるIPアドレスは何ですか?
キャンペーンは、IPホワイトリストに使用するIPアドレスのセットを確立しました。リストを表示するには、Eloqua 接続を編集し、 Eloqua でキャンペーン データを表示できませんか?リンクをクリックして、IP アドレスのリストを表示します。
