キャンペーントラッキング統合を使用する場合、キャンペーンモジュールを使用してビデオを選択し、そのビデオのアクティビティを表示できます。これにより、特定のビデオを視聴したユーザーを確認できます。

キャンペーン対応プレイヤーのすべてのビデオビューが追跡されます。ビデオ アクティビティを表示するには、 をクリックします。ビデオ (非 MAP)左のナビゲーションにあります。[ViewingActivity]ページには、選択した期間のキャンペーン追跡連動で表示されたすべてのビデオのリストが表示されます。の接続を選択コントロールを使用して、特定の接続を選択できます。カレンダーコントロールを使用して、新しい日付範囲を選択できます。列見出しをクリックすると、データを並べ替えることができます。

注意:
  • 表示データが表示されるまでに数時間遅れます。
  • 独立したプラグインとバックエンドシステムによる分析とキャンペーンの追跡同じ結果が見られないのは非常に一般的です。キャンペーンはブラウザの設定を追跡しないといったことも守ります。最も完全な分析については、Analytics モジュールのレポートを参照してください。
  • ビデオアクティビティは、非MAP（キャンペーン追跡）接続でのみ使用できます。

動画のリンクをクリックすると、その動画のすべての視聴アクティビティが表示されます。列見出しをクリックすると、データを並べ替えることができます。

注:ビューアアクティビティは、非MAP（キャンペーントラッキング）接続でのみ使用できます。

その他のデータ表示オプション

キャンペーンモジュールには、データを表示/エクスポートする方法がいくつかあります。

  • 最近の活動- 過去 24 時間のビデオ ビュー イベント データを表示するために使用されます。詳細については、を参照してください。キャンペーン モジュールでのプレイヤー イベントの表示.
  • 視聴者プロフィール- 視聴者によるビデオ アクティビティの表示に使用されます。詳細については、を参照してください。視聴者のプロフィール情報を表示する.
  • データのエクスポート- ビデオ ビューとリード データのエクスポートに使用されます。詳細については、を参照してください。キャンペーン モジュールからのキャンペーン データのエクスポート.
  • オーディエンス API - Audience API を使用して、ビデオ ビューとリード データをプログラムで取得できます。詳細については、を参照してください。概要：オーディエンス API .
