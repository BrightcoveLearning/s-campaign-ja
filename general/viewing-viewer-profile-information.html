キャンペーン追跡統合を使用する場合、ビューアプロファイルは、ビューアを識別するために使用されたデータに基づいて作成されます。ビューアプロファイルを使用すると、特定のビューアの表示データを表示できます。

キャンペーン対応プレイヤーのすべてのビデオビューが追跡されます。ビデオ アクティビティを表示するには、 をクリックします。閲覧者プロフィール (MAP 以外)左のナビゲーションにあります。[ビューアプロファイル（ViewerProfiles）]ページに、選択した期間のキャンペーントラッキング統合からのすべての表示アクティビティのリストが表示されます。カレンダーコントロールを使用して、新しい日付範囲を選択できます。Viewer の値は、ユーザーの識別に使用された値です。通常、これはリードフォームに記録されたメールアドレスになります。列見出しをクリックすると、データを並べ替えることができます。

注:表示データが表示されるまでに数時間遅れます。

ビューアのリンクをクリックすると、そのビューアのすべての表示アクティビティが表示されます。列見出しをクリックすると、データを並べ替えることができます。

その他のデータ表示オプション

キャンペーンデータは、次の方法で表示/エクスポートできます。

  • 最近の活動- 過去 24 時間のビデオ ビュー イベント データを表示するために使用されます。詳細については、を参照してください。キャンペーン モジュールでのプレイヤー イベントの表示.
  • ビデオ活動- ビデオごとに視聴者のアクティビティを表示するために使用されます。詳細については、を参照してください。ビデオ アクティビティの表示.
  • データのエクスポート- ビデオ ビューとリード データのエクスポートに使用されます。詳細については、を参照してください。キャンペーン モジュールからのキャンペーン データのエクスポート.
  • オーディエンス API - Audience API を使用して、ビデオ ビューとリード データをプログラムで取得できます。詳細については、を参照してください。概要：オーディエンス API .
