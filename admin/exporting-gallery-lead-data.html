ギャラリーリードフォームデータは、ギャラリーエクスペリエンスがリードフォームを使用するように構成されている場合に収集されます。リード データを収集するためのポータル エクスペリエンスの構成については、次を参照してください。ポータル エクスペリエンスでのリード フォームの使用.

キャンペーンモジュールからギャラリーリードをエクスポートする場合、キャンペーンリードフォームを使用してキャプチャされたデータだけがエクスポートされます。カスタムリードフォームからのリードデータは、マーケティング自動化ソフトウェアに保存されます。

注:リードデータをダウンロードできるのは、管理者権限を持つユーザーのみです。

ギャラリーのリードデータをエクスポートするには、次の手順に従います。

  1. を開きます運動モジュール。
  2. 左側のナビゲーションで、ギャラリー リードのエクスポート.
  3. リードをエクスポートするギャラリーエクスペリエンスを選択します。
  4. 入力します開始日と終了日リードのために。
  5. エクスポート形式を選択し、コンマ区切り値またエクセル.
  6. クリック書き出すリードデータをダウンロードします。

エクスポートされたファイルには、次のフィールドが含まれます。

  • 可能なすべてのリードフォームフィールド
    • emailAddress
    • FirstName
    • lastName
    • BusinessPHone
    • 国
    • 会社名
    • 業界
  • 外部 ID - 視聴者を一意に識別するための ID
  • プレイヤーID・リードフォームを表示したプレイヤーID
  • ビデオID・データ収集時に視聴していた動画の動画ID
  • ページの URL - データが収集されたときにページ URL ビューアーがオンになっていました
  • 作成日時- データが収集された日時
  • 経験・データ収集時に表示されていた体験名
